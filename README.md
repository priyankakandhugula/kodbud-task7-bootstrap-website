# Bootstrap Website (Event Page)

This project is a responsive website built using Bootstrap 5. It represents a fictional tech event platform called EventHub.

# Features
Responsive Navbar using Bootstrap
Hero section with heading, description, and call-to-action button
Card-based layout using Bootstrap grid system
Fully responsive design for mobile and desktop
Clean footer section
# Technologies Used
HTML5
Bootstrap 5
CSS3 (optional for small enhancements)
# Learning Outcomes

Through this project, I learned how to build responsive websites quickly using Bootstrap components such as navbar, grid system, cards, and utility classes. I also understood how to reduce custom CSS by using pre-built Bootstrap classes effectively.
